

Connection between keypad and RPi GPIO (Model B Rev 1)
---

Using BCM numbering

*Keypad wire -> BCM GPIO*

8 -> 0
7 -> 4
6 -> 25
5 -> 24
4 -> 23
3 -> 22
2 -> 21
1 -> 18
